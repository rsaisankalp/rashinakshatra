# **App Name**: Sankalpa Siddhi

## Core Features:

- User Details Form: Input form for user details: name, WhatsApp number, date of birth, and time of birth.
- Rashi and Nakshatra Verification: API integration: Fetch rashi and nakshatra details using the provided API endpoint based on user inputs.
- Loading Transition: Loading transition: Display a beautiful transition while waiting for the API response. Show the quote: "Offer your sankalpa and gently enter meditation during puja."
- Information Display: Display verified Rashi and Nakshatra, and also show the quote when the result is available.

## Style Guidelines:

- Primary color: Soft saffron (#F4C430), evoking a sense of spirituality and tradition, complementing the tones found at register.vaidicpujas.in
- Background color: Very light beige (#F5F5DC), maintaining a clean and minimalistic backdrop.
- Accent color: Warm terracotta (#E07A5F), used for interactive elements and highlights to draw attention.
- Body text: 'PT Sans' sans-serif for readability in form fields and general information displays.
- Headlines: 'Playfair' serif for headings, lending an elegant, high-end touch.
- Use simple, line-based icons related to astrology and Vedic rituals.
- A clean, centered layout with plenty of whitespace. Forms and information blocks will have rounded corners.
- Subtle animations during loading states, using smooth transitions to avoid abrupt changes. A calming mandala-style loading animation can fill the time spent waiting for the API call to return.
